Turbulence model options¶

This page lists all available options for turbulence closure models.

Generic Lenght Scale model options¶

GLSModelOptions.c1Float

float: c1 parameter for Psi equations

Default:

1.44

GLSModelOptions.c2Float

float: c2 parameter for Psi equations

Default:

1.92

GLSModelOptions.c3_minusFloat

float: c3 parameter for Psi equations, stable stratification

If compute_c3_minus is True this value will be overriden

Default:

-0.52

GLSModelOptions.c3_plusFloat

float: c3 parameter for Psi equations, unstable stratification

Default:

1.0

GLSModelOptions.closure_name‘k-epsilon’|’k-omega’|’Generic Length Scale’

Name of two-equation closure

Default:

‘k-epsilon’

GLSModelOptions.cmu0PositiveFloat

float: cmu0 parameter

Default:

0.5477

GLSModelOptions.compute_c3_minusBool

bool: compute c3_minus from ri_st

Default:

True

GLSModelOptions.compute_cmu0Bool

bool: compute cmu0 from stability function parameters

If compute_cmu0 is True, this value will be overriden

Default:

True

GLSModelOptions.compute_galperin_climBool

bool: compute c_lim length scale limiting factor

Default:

True

GLSModelOptions.compute_kappaBool

bool: compute von Karman constant from schmidt_nb_psi

Default:

False

GLSModelOptions.compute_len_minBool

bool: compute min_len from k_min and psi_min

Default:

False

GLSModelOptions.compute_psi_minBool

bool: compute psi_len from k_min and eps_min

Default:

False

GLSModelOptions.compute_schmidt_nb_psiBool

bool: compute psi Schmidt number

Default:

True

GLSModelOptions.diff_minPositiveFloat

float: minimum value for eddy diffusivity

Default:

1e-08

GLSModelOptions.eps_minPositiveFloat

float: minimum value for epsilon

Default:

1e-14

GLSModelOptions.f_wallFloat

float: wall function parameter

Default:

1.0

GLSModelOptions.galperin_climPositiveFloat

float: Galperin length scale limitation parameter

Default:

0.3

GLSModelOptions.k_minPositiveFloat

float: minimum value for turbulent kinetic energy

Default:

1e-06

GLSModelOptions.kappaFloat

float: von Karman constant

If compute_kappa is True this value will be overriden

Default:

0.4

GLSModelOptions.len_minPositiveFloat

float: minimum value for turbulent length scale

Default:

1e-12

GLSModelOptions.limit_epsBool

bool: apply Galperin length scale limit on epsilon

Default:

False

GLSModelOptions.limit_lenBool

bool: apply Galperin length scale limit

Default:

False

GLSModelOptions.limit_len_minBool

bool: limit minimum turbulent length scale to len_min

Default:

True

GLSModelOptions.limit_psiBool

bool: apply Galperin length scale limit on psi

Default:

True

GLSModelOptions.mFloat

float: parameter m for the definition of psi

Default:

1.5

GLSModelOptions.nFloat

float: parameter n for the definition of psi

Default:

-1.0

GLSModelOptions.pFloat

float: parameter p for the definition of psi

Default:

3.0

GLSModelOptions.psi_minPositiveFloat

float: minimum value for psi

Default:

1e-14

GLSModelOptions.ri_stFloat

steady state gradient Richardson number

Default:

0.25

GLSModelOptions.schmidt_nb_psiPositiveFloat

float: psi Schmidt number

Default:

1.3

GLSModelOptions.schmidt_nb_tkePositiveFloat

float: turbulent kinetic energy Schmidt number

Default:

1.0

GLSModelOptions.stability_function_name‘Canuto A’|’Canuto B’|’Kantha-Clayson’|’Cheng’

Name of stability function family

Default:

‘Canuto A’

GLSModelOptions.visc_minPositiveFloat

float: minimum value for eddy viscosity

Default:

1e-08
